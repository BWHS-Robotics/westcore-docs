std::tuple<float, float> getOutput(const JoystickValue &left, const JoystickValue &right)

Calculates the left and right motor powers for the chassis given joystick values. The output will differ depending on the ChassisConfiguration.

This method will only return the simulated power output and has no effect on the chassis.

Parameters

  • left – The current JoystickValue for the left joystick

  • right – The current JoystickValue for the right joystick

Returns

void update(const JoystickValue &left, const JoystickValue &right)

Calculates and applies motor power to the chassis given joystick value inputs. This should be called in the user control while loop.

Parameters

  • left – The current JoystickValue for the left joystick

  • right – The current JoystickValue for the right joystick

struct JoystickValue

Represents the current x and y values of a joystick, ranging from [-1, 1] in okapi, or [-127, 127] in PROS.
