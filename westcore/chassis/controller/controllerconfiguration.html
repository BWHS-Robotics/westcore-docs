ControllerConfiguration

class westcore::ControllerConfiguration

Represents a controller configuration for driving the robot chassis

Public Types

enum FunctionType

Determines the mathematical function the controller values [-127, 127] will be plugged into

Default = standard linear joystick Cubic = represented by (x^3)/(127^2), allowing less power towards the center of the joystick with a large ramp up. Often used when high precision is needed while still allowing full speed

Values:

enumerator Default
enumerator Cubic
enum JoystickType

Determines the type of joystick drive (chassis only) the controller will be set to

ArcadeRight = 1 joystick control on the right joystick ArcadeLeft = 1 joystick control on the left joystick Tank = 2 joystick control, with each joystick moving an individual side of the robot SplitArcade = 2 joystick control with left = turning only, righht = linear travel only

Values:

enumerator ArcadeRight
enumerator ArcadeLeft
enumerator Tank
enumerator SplitArcade
