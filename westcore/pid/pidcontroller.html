PIDController

class westcore::PIDController

Public Functions

PIDController(const std::function<double()> &currentFunction, PIDGains gains)

Initializes a new PID Controller given a current function and gains.

Parameters

  • currentFunction – The function the PID controller will be pulling the current value from. This might be the current encoder position of the chassis or angle of the IMU.

  • gains – The gains are the constants used in any PID Controller, along with the max windup and kLimit to prevent kI causing too much rampup.
